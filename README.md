# Login page

This repo make simple Login page and can create account or login account

### Tech Stack
+ React Js
+ Tailwind CSS
+ TypeScript
+ Express
+ Postgres

## How to run server 
### `nodemon server.js` 

## How to run website
### `npm run dev` => previously entered into the bookshelf folder
